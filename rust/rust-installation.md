# RUST: Installation

## Installation on macOS/Linux

```bash
$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

If on macOS, install the C compiler with the following:

```bash
$ xcode-select --install
```

## Troubleshooting

Check version with:

```bash
$ rustc --version
```

## Updating and Uninstalling

```bash
$ rustup update
```

```bash
$ rustup self uninstall
```

## Local Documentation

```bash
$ rustup doc
```
